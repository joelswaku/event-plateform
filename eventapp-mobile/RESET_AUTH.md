# Reset Authentication in Expo App

If you're stuck at a screen and can't see the login:

## Option 1: Clear App Data
1. Close the Expo app completely
2. In Expo Go, shake device
3. Select "Clear app data"
4. Restart the app

## Option 2: Reinstall
1. Delete the app from Expo Go
2. Scan QR code again
3. App will start fresh

## Option 3: Manual Logout (if you can navigate)
1. Go to Profile/Settings
2. Click Logout
3. Return to login screen
